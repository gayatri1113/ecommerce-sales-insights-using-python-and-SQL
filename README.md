# ecommerce-sales-insights-using-python-and-SQL
End-to-end E-commerce Data Analysis project using SQL and Python to extract insights on customer behavior, sales trends, and business growth.

This project analyzes real-world e-commerce sales data to uncover insights about 
customer behavior, sales performance, revenue trends, and top performing products.
Using **SQL** for data extraction and **Python (Pandas, Matplotlib, Seaborn)** 
for data cleaning, visualization, and analysis.
